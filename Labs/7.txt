/*ƒана последовательности вещественных чисел и количество членов в последовательности. ¬ычислить сумму чисел, кратных 5 в последовательности. */

#include<stdio.h>
#include<math.h>
main()
{
int i, sum = 0, kol;

printf("Vveite kolichestvo chisel: \n");
scanf("%d", &kol);

int chisla[kol];

printf("Vvedite chisla: \n");
for (i = 0;i<kol;i++)
scanf("%d", &chisla[i]);

for (i = 0;i<kol;i++)
{
if (chisla[i] % 5 == 0)
sum += chisla[i];
}

printf("Summa chisel, kratnih 5 = %d", sum);

getchar();
return 0;
}
